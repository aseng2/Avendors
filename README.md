# Avendors
This is the CPSC 362 Projects
Function Requirement:
1. Tracks the expiration date for every item in the vending machine
2. Able to determine if a recall has been issued for items in vending machines
3. Tracking how many of what good were sold on each day
4. Can tell restocker where each item goes (Locations of snacks)
5. Tracks inventory (Counts how many items)
6. Display inventory to Customer
7. Correct amount of changes to customers
8. Purchase history for management
9. Sends notification for restock
10. Send maintains alert (status update)
